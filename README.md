## Задание 1. На js

Сравнить время ответа разных сервисов:
- yandex.ru
- google.com
- yahoo.com
- avito.ru
- ...

Написать функцию, принимающую на вход массив URL и возвращающую массив объектов, отсортированный в порядке возрастания времени ответа:
``` java-script
[
	{ url1, responseTime1},
	{ url2, responseTime2},
	{ url3, responseTime3},
	...
]

responseTime1 < responseTime2
```

#### Важно
Запросы к сервисам должны выполняться параллельно, но не более чем 3 запроса одновременно

## Задание 2. На верстку

В качестве задания на верстку предлагаем вам сверстать два блока из предоставленного  [макета](https://readymag.com/nagradion/1073740/?)

Решением должна быть предусмотрена возможность изменять группу первенства, отображая при этом соответствующий контент блока

По клику на видео должна открываться модалка со стандартным проигрывателем видео с YouTube

**Задание должно быть выполнено с помощью библиотеки React и корректно отрабатывать в последних версиях браузеров chrome, firefox и safari**

Будет несомненным плюсом:
- использование кастомного конфига webpack
- linting
- unit тесты
- осмысленные коммиты
